MongoDB

MongoDB, the most popular NoSQL database, is an open-source document-oriented database. The term ‘NoSQL’ means ‘non-relational’. It means that MongoDB isn’t based on the table-like relational database structure but provides an altogether different mechanism for storage and retrieval of data. This format of storage is called BSON ( similar to JSON format). A simple MongoDB document Structure: 

            {
                title: 'MongoDB',
                by: 'Mayank Bhugra',
                url: 'https://www.geeksforgeeks.org',
                type: 'NoSQL'
            }
